Plano de aula digital - CONSTRUÇÕES GEOMÉTRICAS / SÓLIDOS GEOMÉTRICOS

Assunto: Poliedros

Componente curricular: Matemática

Etapa: 9° ano


PASSO 1:

Após a aula de revisão sobre Polígonos Regulares / Poliedros, dividir as turmas em equipes de 2 a 3 alunos. Logo após, entregar um roteiro de aula com breves revisões sobre os temas abordados durante as aulas. Como sugestão, pode-se estar utilizando o software matemático PolyPro que utiliza-se de realidade virtual para visualização das planificações de cada sólido geométrico. Isso ajudará durante o contexto da aula na apresentação dos conceitos didáticos.


PASSO 2:

Agora será necessário entregar para cada equipe, folhas com os modelos de sólidos geométricos planificados.

O arquivo de sólidos geométricos para imprimir, recortar e colar está disponível clicando aqui


PASSO 3:

Cada equipe deverá recortar cuidadosamente cada planificação, e em seguida colar seguindo exatamente as instruções contidas em cada desenho. (linhas tracejadas devem ser sobrepostas durante a colagem)
